# FLEVA Platform

A structured student platform designed to improve campus collaboration, communication, and problem-solving. This project serves as a centralized hub for students to connect, share resources, and resolve campus issues efficiently.

## Project Overview

FLEVA was built as part of the Agile Development coursework. It aims to replace scattered communication channels with a single, organized platform featuring specific modules for different student needs.

## Key Features

### 1. Anonymous Posts
A safe space for students to share experiences, concerns, and thoughts without revealing their identity.
- Moderators can forward serious issues to the Problem Hub.
- Options to filter by Trending, Newest, and Most Replies.

### 2. Discussion Rooms
Focused discussion groups for specific batches, clubs, and project teams.
- Organizes conversations into specific categories (Batches, Clubs, Projects).
- Real-time updates simulation.

### 3. Problem Hub
A formal ticketing system for reporting campus facilities and academic issues.
- Track issues through statuses: **New, In Review, Resolved**.
- Priority levels (High, Medium, Low) for urgent attention.
- Integration with Anonymous Posts.

### 4. Skill Exchange
A peer-to-peer learning network.
- Students can offer to teach skills (e.g., React, UI/UX).
- Students can request help or mentorship from others.

### 5. Notes & Resources
A digital library for academic resource sharing.
- Upload and download lecture notes, past papers, and summaries.
- Searchable by course name or file type (PDF, DOCX).

### 6. Idea Incubator
A voting-based system for campus improvements.
- Students submit innovative ideas for the campus.
- The community votes to approve or review proposals.

## Tech Stack

- **Frontend:** React.js
- **UI Library:** Material UI (MUI)
- **Routing:** React Router DOM
- **Icons:** Material UI Icons

## Getting Started

Follow these instructions to set up the project locally.

### Prerequisites

Ensure you have **Node.js** installed on your machine.

### Installation Steps

1. **Clone the repository:**
   
   git clone [https://github.com/Chanii2024/fleva-platform.git](https://github.com/Chanii2024/fleva-platform.git)


2. **Go to the project root directory:**
   
cd fleva-platform-website


3. **Install dependencies:**
   
npm install


4. **Launch the application:**
   
npm start
